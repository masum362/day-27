# 📘 JavaScript Practice Tasks - Simple Functions

## 🎯 Objective

Practice writing simple JavaScript functions using:

* Function declaration
* Parameters
* `return` statement
* Conditional statements (`if...else`)
* Loops
* Arrays
* Strings

---

## 🧩 Task 1: Multiply Four Numbers

Create a function that:

* Takes **four numbers** as parameters.
* Multiplies all four numbers.
* Returns the final result.

### Example

```js
Input: 2, 3, 4, 5
Output: 120
```

---

## 🧩 Task 2: Odd or Even Calculation

Create a function that:

* Takes a single number as a parameter.
* If the number is **odd**, multiply it by **2** and return the result.
* If the number is **even**, divide it by **2** and return the result.

### Example

```js
Input: 7
Output: 14
```

```js
Input: 8
Output: 4
```

---

## 🧩 Task 3: Calculate Average

Write a function named **`make_avg()`**.

The function should:

* Take an array of integers as a parameter.
* Calculate the average of all the numbers in the array.
* Return the average value.

### Example

```js
Input: [10, 20, 30, 40]
Output: 25
```

---

## 🧩 Task 4: Count Zeros

Write a function named **`count_zero()`**.

The function should:

* Take a **binary string** as a parameter.
* Count how many **`0`** characters are present.
* Return the total count.

> **Note:** A binary string contains only `0` and `1`.

### Example

```js
Input: "1010010"
Output: 4
```

---

## 🧩 Task 5: Even or Odd

Write a function named **`odd_even()`**.

The function should:

* Take an integer as a parameter.
* Return **`"Even"`** if the number is even.
* Return **`"Odd"`** if the number is odd.

### Example

```js
Input: 12
Output: "Even"
```

```js
Input: 17
Output: "Odd"
```

---

## 📌 Rules

* ✅ Use **Function Declaration**.
* ✅ Use the **`return`** statement where necessary.
* ✅ Write clean and readable code.
* ❌ Do not use built-in methods that directly solve the problem (unless instructed).
* 💡 Test each function with multiple input values before submitting.
