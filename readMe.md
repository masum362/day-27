# 📘 JavaScript Practice Tasks - Functions (Part 2)

## 🎯 Objective

Practice JavaScript Functions by solving the following problems.

---

## 🧩 Task 1: Find the Smallest Number

Write a function named **`find_smallest()`**.

The function should:

* Take an array of numbers as a parameter.
* Find the smallest number in the array.
* Return the smallest number.

### Example

```js
Input: [12, 45, 7, 23, 56]
Output: 7
```

---

## 🧩 Task 2: Count Vowels

Write a function named **`count_vowels()`**.

The function should:

* Take a string as a parameter.
* Count how many vowels (`a`, `e`, `i`, `o`, `u`) are present.
* Return the total number of vowels.

### Example

```js
Input: "javascript"
Output: 3
```

---

## 🧩 Task 3: Reverse an Array

Write a function named **`reverse_array()`**.

The function should:

* Take an array as a parameter.
* Return a new array with the elements in reverse order.
* **Do not use the built-in `reverse()` method.**

### Example

```js
Input: [1, 2, 3, 4, 5]
Output: [5, 4, 3, 2, 1]
```

---

## 🧩 Task 4: Count Positive Numbers

Write a function named **`count_positive()`**.

The function should:

* Take an array of integers as a parameter.
* Count how many numbers are greater than `0`.
* Return the total count.

### Example

```js
Input: [-5, 10, 0, 8, -2, 15]
Output: 3
```

---

## 🧩 Task 5: Remove Duplicate Values

Write a function named **`remove_duplicate()`**.

The function should:

* Take an array as a parameter.
* Return a new array containing only unique values.
* **Do not use `Set`.**

### Example

```js
Input: [1, 2, 2, 3, 4, 4, 5]
Output: [1, 2, 3, 4, 5]
```

---

## 📌 Rules

* ✅ Use **Function Declaration**.
* ✅ Use loops (`for` or `while`) where necessary.
* ✅ Return the result from each function.
* ❌ Do not use built-in methods that directly solve the problem (`reverse()`, `Set`, etc.).
* 💡 Test each function with different input values.
